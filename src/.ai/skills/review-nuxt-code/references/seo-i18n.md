# SEO and Internationalization Review

- Verify indexable primary content exists in server-rendered HTML.
- Check title, description, canonical, social metadata, and structured data match visible content.
- Verify JSON-LD contains real, supported, and current facts.
- Check robots directives, sitemap behavior, and canonical URLs against the environment policy.
- Verify locale routing, default locale, and fallback behavior follow configured project decisions.
- Check `hreflang` values are reciprocal and point to valid localized canonical pages when used.
- Verify the document language and localized metadata match the rendered locale.
- Check user-facing strings follow the configured translation strategy.
- Do not require AI-specific markup or files without an explicit platform requirement.
